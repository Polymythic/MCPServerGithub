# MCPServerGithub

A FastAPI-based MCP server for interactions with GitHub.

## Setup

1. Install [uv](https://github.com/astral-sh/uv):
   ```sh
   pip install uv
   ```
2. Install dependencies:
   ```sh
   uv pip install -r pyproject.toml
   ```
3. Run the server:
   ```sh
   uvicorn src.server:app --reload
   ```
